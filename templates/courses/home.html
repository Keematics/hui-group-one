About This Project

Project Overview

This is a comprehensive learning management system built as a group assignment. The platform enables users to:

  • Register and manage their learning profiles
  • Access professional certifications with multiple courses
  • Learn through diverse content types (text, images, videos)
  • Track progress across all courses and modules
  • Earn certificates upon course completion (90% threshold)
  • Download professional certification certificates
Technology Stack
Django
MySQL
jQuery
Cloudinary
Key Features
  • Progress Tracking: Video modules require 85% completion, while text/image modules need manual confirmation
  • Certificate Generation: Automatic PDF certificate generation upon achieving 90% course completion
  • Professional Certifications: Complete all courses in a certification to earn a comprehensive professional certificate
  • Responsive Design: Fully responsive interface that works seamlessly across all devices
